# 🌐 Polyglot Lab

> **Explore the world of programming languages through one interactive playground.**

Polyglot Lab is a modern and visually engaging web application that lets users discover programming languages in a fun and informative way. Simply select any language to explore its history, creator, syntax, advantages, disadvantages, salary insights, popular companies, and evolution timeline through beautiful animated cards.

🔗 **Live Demo:** https://panthshah2007.github.io/Polyglot-Lab/

---

## ✨ Features

- 🌍 Interactive Programming Language Explorer
- 👨‍💻 Learn about language creators and history
- 📝 Syntax examples for every language
- 💼 Popular companies using each language
- 💰 Salary insights
- ✅ Pros and Cons comparison
- 📈 Evolution timeline
- 🎨 Smooth animations and modern UI
- 📱 Fully responsive design
- ⚡ Fast and lightweight

## 🌟 Why Polyglot Lab?

Instead of searching different websites for language information, Polyglot Lab brings everything together in one beautiful interface.

It provides:
- History
- Creator
- Syntax
- Salary
- Companies
- Pros & Cons
- Timeline

—all accessible with a single click.
